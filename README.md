# Sobre o Projeto - Livraria Alta Cultura
**Descrição**: O projeto “Livraria Alta Cultura” nasceu com o intuito de ser uma biblioteca, mas uma biblioteca que contivesse apenas alguns dos nossos intelectuais brasileiros, sendo assim uma biblioteca voltada para a vida intelectual do sujeito.

# ⌨Tecnologias utilizadas
1. HTML
2. CSS
3. JS
4. Bootstrap

# Qual a finalidade desse projeto?
Esse projeto tem como finalidade demonstrar o conhecimento que possuímos sobre HTML, CSS, JS e Bootstrap. Não é um projeto com fins comerciais, mas sim voltado ao aprendizado e à aquisição de experiência por meio de seu desenvolvimento.

# 🧮Qual foi o nosso maior problema e como solucionamos ele?
Nosso maior problema foi efetivar a funcionalidade do Bootstrap no código, pois estávamos colocando o link de estilização dele depois do link de estilização principal, que era o do CSS. Para resolver isso, colocamos o link de estilização do Bootstrap antes do link do CSS que já estava naquele campo, e então o Bootstrap foi efetivado e funcionou com sucesso.
