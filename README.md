# Sobre o Projeto - Livraria Alta Cultura
**Descrição**: O projeto "Livraria Alta Cultura", nasceu com o intuito de ser um projeto de biblioteca, mas uma biblioteca que contive-
sse apenas alguns dos nossos intelectuais que foram brasileiros, sendo assim uma biblioteca com finalidade para a vida intelectual do 
sujeito.

# ⌨Tecnologias utilizadas
1. HTML
2. CSS
3. JS
4. Bootstrap

# Qual a finalidade desse projeto?
Esse projeto tem como finalidade demonstrar o conhecimento que possuímos sobre HTML, CSS, JS e Bootstrap, não é um projeto para fim com-
ercial, mas para um fim de aprendizado e experiência mediante o projeto.

# 🧮Qual foi o nosso maior problema e como solucionamos ele?
Nosso maior problema foi efetivar a funcionalidade Bootstrap no código, pois estavamos colocando o link de estilização dele depois do link de
estilização primário que era o do CSS, então para resolvermos isso colocamos o link de estilização do Bootstrap antes do link do CSS que já tinha
naquele campo e o Bootstrap foi efetivado e funcionou com sucesso.
